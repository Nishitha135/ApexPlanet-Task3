# ApexPlanet Internship - Task 3

## 📌 Project Overview
This project was developed as part of the ApexPlanet Web Development Internship. It enhances a PHP CRUD Blog Application by adding advanced features such as search functionality, pagination, and an improved responsive user interface.

## 🎯 Objective
- Add search functionality to search posts by title or content.
- Implement pagination for efficient navigation.
- Improve the user interface and user experience.
- Create a responsive web application using modern web technologies.

## ✨ Features
- 🔐 User Registration & Login
- 📝 Create, Read, Update & Delete (CRUD) Blog Posts
- 🔍 Search Posts by Title or Content
- 📄 Pagination
- 🎨 Responsive User Interface
- 💾 MySQL Database Integration

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Bootstrap

## 🚀 How to Run
1. Install XAMPP.
2. Copy the project folder to the `htdocs` directory.
3. Start Apache and MySQL from the XAMPP Control Panel.
4. Import the database into phpMyAdmin.
5. Open the project in your browser.
