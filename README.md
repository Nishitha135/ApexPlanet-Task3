# ApexPlanet Internship - Task 3

## 📌 Project Overview
This project is developed as part of the ApexPlanet Web Development Internship. The application enhances a PHP CRUD Blog with advanced features such as search functionality, pagination, and an improved responsive user interface.

## 🎯 Objective
- Add search functionality to find posts by title or content.
- Implement pagination for efficient post navigation.
- Improve the user interface and user experience.
- Build a responsive design using HTML, CSS, JavaScript, and Bootstrap.

## ✨ Features
- 🔐 User Registration & Login
- 📝 Create, Read, Update & Delete (CRUD) Blog Posts
- 🔍 Search Posts by Title or Content
- 📄 Pagination for Post Listings
- 🎨 Responsive and Modern UI
- 💾 MySQL Database Integration

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Bootstrap

## 🚀 How to Run
1. Install XAMPP.
2. Copy the project folder to `htdocs`.
3. Start Apache and MySQL.
4. Import the database into phpMyAdmin.
5. Open the project in your browser:
   ```
   http://localhost/Task3/
   ```
